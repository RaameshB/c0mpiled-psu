# Git Conventions

- Commit frequently, conventional commits, as you produce work
- Commits: `feat(scope): description`, `fix(scope): description`,
  `refactor(scope): description`
- Branches: Use kebab-case without prefixes (e.g.,
  `vehicle-chassis-model-overhaul`, not `feat/vehicle-chassis-model-overhaul`)
